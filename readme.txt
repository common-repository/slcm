== Description ==
Skype Live Chat Messenger – Now that Microsoft bought skype and officially phased out live messenger, it was time to launch the ultimate skype communication tool for your wordpress. The most cost effective Live Chat plugin. Chat with your visitors for free! 

Plugin Features: 

1. You Can Change the button of your Chat box in to 3, Rectangle, Bubble and Rounded
2. You Can Change the Color of your Button Chat box.
3. You Can Change the Color of your Message.
4. You Can Put a text label on the Rectangle and Rounded button.

== Installation ==

This section describes how to install the plugin and get it working.
1. Download and install the plugin from WordPress dashboard. You can also upload the entire "Download Skype Live Chat Messenger" folder to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Done
